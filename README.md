# 使用说明

##### 初始化 conda for PowerShell
```bash
conda init powershell
```


##### 创建新环境
```bash
conda create --name Cloudflare python=3.10.16 -y
```


##### 激活环境
```bash
conda activate Cloudflare
```


##### 依赖安装
```bash
pip install -r requirements.txt
```


##### 运行项目
```bash
python test.py
```
